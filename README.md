# UX Transformation Case Study — Christopher Kaye

A psychology-driven portfolio case study built with Vite + React.

## Features
- Animated metric counters (scroll-triggered)
- Interactive UX Law constellation map with side-sheet detail panels
- Before/After platform comparisons
- Scroll-triggered phase timeline with expandable drawers
- Sticky progress bar + active section nav
- Mobile-first hamburger nav
- Portal-based tooltips (zero clipping)
- UX Maturity progression bars

## Deploy

### Via Vercel (recommended)
1. Push this repo to GitHub
2. Go to vercel.com → New Project → Import from GitHub
3. Framework: Vite (auto-detected)
4. Deploy — done

### Local dev
```bash
npm install
npm run dev
```

### Build
```bash
npm run build
```
